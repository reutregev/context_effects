Each file contains processed responses of the experiment.
Each row in a file represents a response of a participant to a trial in the experiment.

Columns:
* ResponseId - A unique ID assigned to a participant
* condition - The experiment condition a participant was assigned to
* choice_set - The choice set that was displayed and to which the questions were refer in the current experiment trial 
* choice_set_size - Specifying the form of the displayed choice set, i.e., pair (p) or triplet (t)
* choice - Indication of the participant's answer in the choice question (1=target alternative, 0=competitor or decoy alternative)
* scale_target_normalized - The scale value for the target alternative, normalized per participant
